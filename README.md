# Simple-POS-for-WooCommerce
A simple WooCommerce POS plugin that allows store admins to manage quick sales, products, carts, and checkout directly from a custom WordPress dashboard.
Simple POS for WooCommerce is a custom WordPress plugin that adds a lightweight point-of-sale system inside the WordPress dashboard. It helps store admins quickly search or select WooCommerce products, add them to a cart, adjust quantities, and create customer orders without using the standard storefront checkout.

This plugin is useful for retail stores, small businesses, and WooCommerce shop owners who want a simple in-dashboard POS solution.
